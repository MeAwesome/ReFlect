## Installation

This module comes pre-installed with ReFlect.

## Configuration

| Option | Description
| ------ | ------
| `region` | The region on the mirror to be displayed. <br><br> **Possible Values:** *Read Mirror >  Regions* <br> **Default Value:** `top-left`
